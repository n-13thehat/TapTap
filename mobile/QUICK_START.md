# 🚀 TapTap Mobile - Quick Start Guide

## ⚡ Get Running in 60 Seconds

### 1. Install & Start
```bash
cd mobile
npm install
npm start
```

### 2. View on Device
- Install **Expo Go** on your phone
- Scan the QR code
- App loads instantly!

## 🎯 Test These Features

### Music Player
1. Go to Music tab
2. Tap any track
3. **Test:**
   - Play/pause/skip
   - Tap "Lyrics" → See synchronized lyrics
   - Tap "Queue" → Manage queue
   - Adjust volume
   - Toggle shuffle/repeat

### Social Feed
1. Go to Social tab
2. **Test:**
   - Scroll through posts
   - Like posts (heart icon)
   - Filter by All/Following/Trending
   - Pull to refresh

### Battles
1. Go to Battles tab
2. **Test:**
   - View active battles
   - Vote on tracks
   - See real-time percentages
   - Filter by Active/Upcoming/Ended

### Marketplace
