## 🎯 Quick Commands

### Start Development
```bash
cd mobile
npm start
```

### Install Dependencies
```bash
npm install
```

### Clear Cache & Start
```bash
npx expo start --clear
```

### Build for Production
```bash
eas build --platform all
```
