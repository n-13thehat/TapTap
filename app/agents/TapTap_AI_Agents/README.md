# TapTap AI Agents

- **agents/** — JSON profiles (source of truth)
- **profiles/** — Markdown briefs (human-readable)
- **sprints/** — Upgrade scripts you can run anytime
- **backups/** — Auto snapshots when you re-run the one-shot

## Quick Start
1) Run: .\sprints\Run_All_Sprints.ps1
2) Re-open any JSON/MD to see upgrades applied.
3) Re-run sprints anytime; they are idempotent.
